# ascii-art-web

## Description
ascii-art-web is a web application that allows users to generate ASCII art from text using different banner styles. The project is built in Go and provides a simple web interface where users can type text, choose a banner style, and see the ASCII art result instantly.

Supported banners:
- `standard`
- `shadow`
- `thinkertoy`

The project uses Go's `net/http` package and HTML templates to render the web page and display results.
